Gerador de Arranjo de Tiles

Descrição dos Parâmetros:

Número de Espirais: Controla o número de espirais no arranjo.

Tiles por Espiral: Define quantos tiles serão dispostos em cada espiral.

Raio Base (x tile_size): O raio inicial do arranjo, medido a partir do centro até o início das espirais. Multiplicado pelo tamanho do tile.

Fator de Crescimento do Raio (x tile_size): Fator pelo qual o raio aumenta a cada novo tile dentro de uma espiral. Multiplicado pelo tamanho do tile.

Fator de Offset do Ângulo: Determina o ângulo inicial para a disposição dos tiles em cada espiral.

Fator de Variação do Ângulo: Adiciona uma variação angular adicional para cada tile, permitindo curvas mais complexas nas espirais.

Fator Exponencial do Raio: Aplica um crescimento exponencial ao raio, fazendo com que a distância entre os tiles aumente de forma não linear.

Fator Exponencial do Ângulo: Introduz uma variação exponencial no ângulo, alterando a curvatura das espirais de forma não linear.

Rotação por Espiral: Aplica uma rotação adicional a cada espiral subsequente, criando um efeito de torção no arranjo.

Tile Central: Adiciona um tile no centro do arranjo, servindo como ponto de referência.

Nome da Estação: Um nome descritivo para o arranjo gerado.
